Warum Java lernen?

Java ist eine der vielseitigsten und gefragtesten Programmiersprachen in der heutigen Softwareentwicklung. Mit Java könnt ihr Desktop-Anwendungen, Web-Anwendungen, mobile Apps und sogar serverseitige Anwendungen erstellen. Es ist eine Sprache, die in der Industrie weit verbreitet ist und euch viele berufliche Möglichkeiten bietet.

Für wen ist dieser Kurs geeignet?

Unser Java-Kurs richtet sich sowohl an Anfänger als auch an fortgeschrittene Studierende. Egal, ob ihr noch nie zuvor programmiert habt oder bereits Grundkenntnisse in Java besitzt, dieser Kurs bietet euch die Möglichkeit, euer Wissen zu vertiefen und eure Fähigkeiten in der Java-Programmierung zu erweitern.

Was werden wir lernen?

In diesem Kurs werden wir von den grundlegenden Konzepten der Java-Programmierung bis zur fortgeschrittenen Anwendungsentwicklung geführt. Ihr werdet die Sprache beherrschen, objektorientierte Programmierung verstehen und lernen, wie ihr praktische Anwendungen erstellen könnt. Wir werden Themen wie Datenstrukturen, Fehlerbehebung und Best Practices behandeln, um sicherzustellen, dass ihr in der Lage seid, qualitativ hochwertigen Java-Code zu schreiben. Der Kurs wird folgende Themen beinhalten:

A. Imperative Programmierung:
  • 1. Grundlagen:
      Java Syntax, Kommentare, Variablen, Literale (Werte), Datentypen, Operatoren, Casting
  • 2. Flusskontrolle:
      If-else Statements, Switch-Anweisung, For-Schleife, While-Schleife, Break, Continue
  • 3. Weitere Themen:
      Zeichenketten, Arrays, Methoden, Input & Output, Random Klasse, Error Handling (try, catch, throw)
B. Objektorientierte Programmierung (OOP):
  • 1. Klassen & Objekte:
      Syntax, Static Modifikator, Klassen/Instanz-Variablen, Klassen/Instanz-Methoden, Konstruktoren
